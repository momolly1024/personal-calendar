# Personal Calendar

![](https://i.imgur.com/9kTuLb3.png)

use React / Python / Google sheet to write a personal calendar

## HOW TO USE

1. git clone
2. cd /frontend

-   yarn install
-   yarn start

3. cd /backend

-   python3 -m venv env
-   source env/bin/activate
-   pip install -r requirements.txt
-   uvicorn main:app --reload

4. change key.js
5. change calendar.json
